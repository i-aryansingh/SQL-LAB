# Step 3: Create EMPLOYEE Table
### Query
```sql
CREATE TABLE EMPLOYEE (
    EMPNO INT(4) PRIMARY KEY,
    ENAME VARCHAR(20) NOT NULL,
    JOB VARCHAR(20),
    MGR INT(4),
    HIREDATE DATE,
    SAL DECIMAL(10,2),
    COMM DECIMAL(7,2),
    DEPTNO INT(2),
    FOREIGN KEY (DEPTNO) REFERENCES DEPARTMENT(DEPTNO)
);
```
### Output  
Table EMPLOYEE created successfully.
